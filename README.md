# 🤖 AI Assistant

An AI-powered chatbot built using **Google Gemini**, **Streamlit**, and **LangGraph**. The application provides an interactive chat interface with persistent chat history and a modern UI.

## 🚀 Live Demo

🌐 https://langgraphchatbot-ycnaskjvfx2nixcgpgmb3w.streamlit.app/

## 💻 GitHub Repository

https://github.com/Dhanush6281/LangGraph_Chatbot

---

## ✨ Features

- 🤖 AI-powered conversations using Google Gemini
- 💬 Persistent chat history
- 🎨 Modern and responsive Streamlit UI
- ⚡ Fast responses
- 🔒 Secure API key management using environment variables
- 🗂️ Sidebar with chat history
- 🧹 Clear chat history option

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- LangGraph
- python-dotenv
- PyPDF

## ⚙️ Installation

```bash
git clone https://github.com/Dhanush6281/LangGraph_Chatbot.git

cd LangGraph_Chatbot

pip install -r requirements.txt

streamlit run streamlit_app.py
```

---

## 🔑 Environment Variable

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
